# 问题反馈

感谢您使用「抵税证书助手」。如果您在使用过程中遇到问题或有任何建议，请通过以下方式联系我们。

## 反馈方式

### 方式一：应用内反馈

在应用中打开「我的」→「帮助反馈」：
1. 选择反馈类型（功能建议、问题反馈、使用疑问、其他）
2. 填写反馈内容
3. 点击「发送邮件反馈」

### 方式二：邮件反馈

直接发送邮件至：**taxhelper@example.com**

邮件格式建议：
- 标题：[抵税助手] 问题类型 - 简短描述
- 内容：详细描述问题或建议

### 方式三：GitHub Issues

如果您有 GitHub 账号，可以在本仓库提交 Issue：
- [提交新 Issue](https://github.com/nx7tq9dk46-web/taxhelper-pub/issues)

---

## 反馈类型说明

| 类型 | 说明 |
|------|------|
| 功能建议 | 您希望应用增加的功能或改进 |
| 问题反馈 | 应用出现的错误、Bug 或异常行为 |
| 使用疑问 | 对应用功能使用方法的疑问 |
| 其他 | 其他任何问题或建议 |

---

## 反馈处理

- 我们会在 **1-3 个工作日** 内回复您的反馈
- 紧急问题会优先处理
- 功能建议会纳入后续版本规划

---

## 反馈内容建议

为了让问题更快解决，请尽量提供以下信息：

**对于 Bug 反馈：**
- 问题发生的步骤
- 您的设备型号和系统版本
- 问题截图（如有）

**对于功能建议：**
- 您期望的功能描述
- 为什么这个功能对您有帮助

---

## 感谢您的反馈！

您的反馈是我们改进应用的重要依据。感谢您的支持！
